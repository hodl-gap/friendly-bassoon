## Case 04: SCOTUS Tariff Ruling (16/20)

### Guideline Answer (Korean analyst + market data)

The SCOTUS 6-3 ruling struck down IEEPA tariffs. The expected analysis:
1. **Rally is short-lived** — substitute tariffs under Section 122 (10%→15%) partially reverse gains
2. **$100-175bn refund** creates two opposing forces: corporate cash infusion (bullish) vs fiscal deficit (bearish)
3. **Disinflation chain**: tariff removal → import prices fall → inflation eases → Fed rate cut justification
4. **Treasury issuance chain**: refund obligation → treasury issuance → yields UP
5. **Rate tension**: yield UP (fiscal) vs yield DOWN (disinflation) — opposing forces
6. **Policy transition**: IEEPA → Sec.122 (150-day limit) → 232/301 (12-18 month investigation)
7. **10Y threshold at 4.2-4.3%** flips refund narrative from bullish to bearish
8. **Sector winners**: non-strategic imports (apparel, toys) and allied-nation imports

## Case 05: Put-Call Ratio (13/16)

1. **Institutions hedging heavily** — put premium >> call premium

4. **Conclusion**: large volatility imminent, NOT a directional signal
5. **Bearish precedents**: Sept 2001 (pre-9/11), 2008 Lehman, 2020 COVID
6. **Bullish precedents**: Jan 2023, March 2022 — high put ratio acted as bottom signal
7. **Key insight**: put surge = "peak fear" signal, but NOT confirmed decline — macro context determines direction

## Case 06: Labor Market Equilibrium (12/16)

### Guideline Answer (BCA Research)

BCA's thesis chain:
1. **Labor demand = supply** for first time post-pandemic → fragile equilibrium
2. **Fed "runs hot"** intentionally — tolerates 2.5-3.5% inflation to prevent recession

4. **Dollar weakens** — real rate differential narrows → USD less attractive
5. **Bear steepening** — inflation expectations push long yields up, short yields capped by easing
6. **Equities outperform bonds** — easing + lower real rates + nominal growth = positive
7. **Tactical rotation**: consumer discretionary oversold vs industrials → overweight consumer, underweight industrials

**5-track insight report:**

## Cross-Case Patterns

### What the Pipeline Consistently Does Well

1. **Web chain extraction** — compensates for DB gaps by finding 17-20 chains per query from Bloomberg, Forbes, Yahoo Finance, WSJ
2. **Data fetching** — comprehensive current data (18-40 variables per run including period-over-period changes)
3. **Multi-track structure** — separates competing narratives into independent tracks with confidence scores
4. **Named source attribution** — cites specific institutions (Goldman, RBC, RSM, Tom Lee) rather than generic claims
5. **Quantitative anchors** — specific numbers throughout ($287B, 2.4% EBIT, 85.6% no-cut probability)

### What the Pipeline Consistently Misses

1. **Forward-looking regime analysis** — anchors on current data rather than projecting regime changes (Fed "run hot", real rate decline)
2. **Deep macro transmission channels** — misses multi-hop chains like real rate → dollar → curve steepening that require integrated macro reasoning
3. **Market microstructure** — dark pool flows, dealer gamma/delta hedging, options market structure
4. **Specific analyst thresholds from non-ingested content** — 10Y at 4.2-4.3% (Hana), consumer vs industrials rotation (BCA)
5. **Classic historical precedents for sentiment indicators** — knows 2022 but misses 2001/2008/2020 as crash precedents for put-call extremes

### Root Cause Analysis

| Pattern | Root Cause | Fix Category |
|---|---|---|
| Missing forward regime analysis | Opus prompt doesn't guide toward "intentional policy regime" reasoning | Prompt engineering |
| Missing macro transmission channels | Derived metrics computed but not surfaced as causal inputs in the Opus prompt | Prompt / data presentation |
| Missing microstructure | No options microstructure data sources ingested | Content coverage |
| Missing analyst thresholds | Specific analyst notes not ingested into Pinecone | Content coverage |
| Missing classic precedents | Web chain extraction LLM picks data-richest source rather than scanning for ALL historical examples | Web chain extraction prompt |
