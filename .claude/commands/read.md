Read the current project folder (NOT the parent folder). Understand and summarize:

1. **Goal/intention** — what does this project do and why?
2. **Structure** — directory tree, major modules, key files
3. **Architecture** — tech stack, pipeline/data flow, how components connect
4. **Current status** — recent git commits, what's working, what's in progress
