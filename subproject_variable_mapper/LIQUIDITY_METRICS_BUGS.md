## How to Use This File

When processing variables through the mapper, if ANY strange or incorrect entries are detected in the liquidity metrics CSV, log them here for later fixing in `subproject_database_manager`.

### Entry Format

```
## [DATE] Issue Description
- **Row**: normalized name or variant
- **Problem**: What's wrong
- **Suggested Fix**: How to fix it
- **Detected During**: Which step/operation found this
```
